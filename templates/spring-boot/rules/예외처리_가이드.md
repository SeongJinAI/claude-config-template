## 2. ErrorCode enum 구조

```java
@Getter
@RequiredArgsConstructor
public enum ErrorCode {

    // ── 비즈니스 코드 ──
    NOT_FOUND("ERR_NOT_FOUND", HttpStatus.NOT_FOUND),
    UNAUTHORIZED("ERR_UNAUTHORIZED", HttpStatus.FORBIDDEN),
    ALREADY_EXISTS("ERR_ALREADY_EXISTS", HttpStatus.CONFLICT),
    INVALID_STATUS("ERR_INVALID_STATUS", HttpStatus.BAD_REQUEST),

    // ── 인프라 코드 (핸들러 전용) ──
    REQUIRE_PARAMETER("ERR_REQUIRE_PARAMETER", HttpStatus.BAD_REQUEST),
    VALIDATION_FAILED("ERR_VALIDATION_FAILED", HttpStatus.BAD_REQUEST),
    DATA_INTEGRITY("ERR_DATA_INTEGRITY", HttpStatus.CONFLICT),
    INTERNAL_SERVER("ERR_INTERNAL_SERVER", HttpStatus.INTERNAL_SERVER_ERROR),
    ;

    private final String code;
    private final HttpStatus httpStatus;
}
```

## 3. BusinessException 사용법

```java
// 엔티티 조회 실패
Entity entity = repository.findById(code)
        .orElseThrow(() -> new BusinessException(ErrorCode.NOT_FOUND));

// 비즈니스 규칙 위반
if (repository.existsByName(name)) {
    throw new BusinessException(ErrorCode.ALREADY_EXISTS);
}

// 상태 검증
if (entity.isDeleted()) {
    throw new BusinessException(ErrorCode.INVALID_STATUS);
```

## 4. GlobalExceptionHandler 템플릿

```java
@Slf4j
@RestControllerAdvice
public class GlobalExceptionHandler {

    // ── 1. 요청 검증 (400) ──

    @ExceptionHandler(MissingServletRequestParameterException.class)
    public ResponseEntity<ApiResponse<Void>> handle(MissingServletRequestParameterException e) {
        log.warn("필수 요청 파라미터 누락: {}", e.getMessage());
        return ResponseEntity.status(HttpStatus.BAD_REQUEST)
                .body(ApiResponse.error(ErrorCode.REQUIRE_PARAMETER.getCode()));
    }

    @ExceptionHandler(MethodArgumentNotValidException.class)
    public ResponseEntity<ApiResponse<Void>> handle(MethodArgumentNotValidException e) {
        String errorCode = e.getBindingResult().getFieldErrors().stream()
                .findFirst()
                .map(FieldError::getDefaultMessage)
                .orElse(ErrorCode.VALIDATION_FAILED.getCode());
        log.warn("요청 데이터 검증 실패: {}", errorCode);
        return ResponseEntity.status(HttpStatus.BAD_REQUEST)
                .body(ApiResponse.error(errorCode));
    }

    // ── 2. 인증/인가 (401, 403) ──

    @ExceptionHandler(BadCredentialsException.class)
    public ResponseEntity<ApiResponse<Void>> handle(BadCredentialsException e) {
        log.warn("인증 실패: {}", e.getMessage());
        return ResponseEntity.status(HttpStatus.UNAUTHORIZED)
                .body(ApiResponse.error(ErrorCode.UNAUTHORIZED.getCode()));
    }

    // ── 3. 비즈니스 로직 ──

    @ExceptionHandler(BusinessException.class)
    public ResponseEntity<ApiResponse<Void>> handle(BusinessException e) {
        ErrorCode errorCode = e.getErrorCode();
        log.warn("비즈니스 예외: {}", errorCode.getCode());
        return ResponseEntity.status(errorCode.getHttpStatus())
                .body(ApiResponse.error(errorCode.getCode()));
    }

    // ── 4. 인프라/시스템 + 모니터링 ──

    @ExceptionHandler(DataIntegrityViolationException.class)
    public ResponseEntity<ApiResponse<Void>> handle(DataIntegrityViolationException e) {
        log.error("DB 정합성 오류: {}", e.getMessage());
        Sentry.captureException(e);
        return ResponseEntity.status(HttpStatus.CONFLICT)
                .body(ApiResponse.error(ErrorCode.DATA_INTEGRITY.getCode()));
    }

    @ExceptionHandler(RuntimeException.class)
    public ResponseEntity<ApiResponse<Void>> handle(RuntimeException e) {
        log.error("예상치 못한 런타임 예외", e);
        Sentry.captureException(e);
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body(ApiResponse.error(ErrorCode.INTERNAL_SERVER.getCode()));
    }
}
```

## 5. 로그 메시지 컨벤션

한글, `"[원인 요약]: {}"` 형식:

| 핸들러 | 로그 메시지 |
|--------|------------|
| `MissingServletRequestParameterException` | `필수 요청 파라미터 누락: {}` |
| `MethodArgumentNotValidException` | `요청 데이터 검증 실패: {}` |
| `BadCredentialsException` | `인증 실패: {}` |
| `AccessDeniedException` | `인가 부족: {}` |
| `BusinessException` | `비즈니스 예외: {}` |
| `DataIntegrityViolationException` | `DB 정합성 오류: {}` |
| `RuntimeException` | `예상치 못한 런타임 예외` (스택트레이스 포함) |

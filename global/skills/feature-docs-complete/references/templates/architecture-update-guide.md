# 아키텍처 설명서 업데이트 가이드

구현 완료 후 아키텍처 설명서를 업데이트할 때 다음 항목을 확인합니다.

## 업데이트 체크리스트

1. **전체 흐름**: 실제 구현된 클래스/메서드명으로 교체
2. **시퀀스 다이어그램**: 구현된 호출 순서와 일치하는지 확인
3. **Entity 관계도**: 실제 연관관계, CASCADE 설정 반영
4. **계층별 책임**: 실제 클래스명과 역할 매핑
5. **테스트 시나리오**: 실제 테스트 케이스 기반으로 보완
6. **권한 구분**: 실제 @PreAuthorize 등 권한 설정 반영
7. **status**: `draft` → `confirmed`로 변경

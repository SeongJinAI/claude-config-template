# 에러 메시지 추가 가이드

이 템플릿은 별도 파일이 아닌 **기존 ERROR_MESSAGES.md에 행을 추가**하는 형식입니다.

## 추가 위치

프로젝트의 `src/docs/ERROR_MESSAGES.md` 파일에서 해당 도메인 섹션을 찾아 추가합니다.
도메인 섹션이 없으면 "로그 전용" 섹션 바로 위에 새 섹션을 생성합니다.

## 섹션 형식

```markdown
## {도메인명}

| 에러 코드 | 의미 | HTTP 상태 |
|-----------|------|-----------|
| `ERR_CODE_NAME` | 에러 설명 | 400/403/404/409 |
```

## ErrorCode enum 동시 추가

`global/exception/ErrorCode.java`에도 동시 추가:

```java
// ── {도메인명} ──
CODE_NAME("ERR_CODE_NAME", HttpStatus.XXX),
```

## 섹션 매핑 기준

| HTTP 상태 | ErrorCode 섹션 |
|----------|---------------|
| 400 | 유효성 검증 오류 / 필수값 누락 / 상태 오류 |
| 403 | 인증/권한 |
| 404 | 존재하지 않음 |
| 409 | 이미 존재함 / 중복 |

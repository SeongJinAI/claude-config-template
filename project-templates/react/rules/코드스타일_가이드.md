# React 코드스타일 가이드

## 컴포넌트

- 함수형 컴포넌트만 사용 (class 컴포넌트 금지)
- 컴포넌트 파일명: PascalCase (`BodyMap.tsx`)
- Props 타입은 `interface`로 정의 (컴포넌트명 + Props)

```tsx
interface BodyMapProps {
  onSelect: (part: string) => void;
}

export function BodyMap({ onSelect }: BodyMapProps) {
  return <div>...</div>;
}
```

## 상태 관리

- 로컬 상태: `useState` / `useReducer`
- 서버 상태: React Query (`useQuery`, `useMutation`)
- 전역 상태: 필요 시 Zustand (최소한으로 사용)

## 스타일링

- Tailwind CSS 유틸리티 클래스 사용
- 인라인 스타일 지양
- 반응형: 모바일 우선 (`sm:` → `md:` → `lg:`)

## 파일 구조 규칙

- 기능별 디렉토리로 구분 (`components/body-map/`, `components/symptom-builder/`)
- 공통 UI는 `components/ui/`에 배치
- 커스텀 훅은 `hooks/`에 배치
- API 호출은 `lib/api/`에 집중

## 타입

- `any` 사용 금지
- API 응답 타입은 `types/`에 정의
- Props 타입은 컴포넌트 파일 내에 정의
